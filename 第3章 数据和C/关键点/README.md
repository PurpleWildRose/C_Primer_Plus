本章先介绍下会用到的三个函数：

## sizeof():返回一种数据类型所占有的字节数。

例如：sizeof(int)  // 基本会返回4，也就是4字节。

## scanf():将用户输入的内容写入到指定变量。

例如： int m; scanf("%d", &m);

虽然不能直接可以写入字符串，但是可以通过std::string str; std::cin >> str; 去写入；亦或者char str[1024];scanf("%s", &str); string bt = str(str);

## getchar():功能与scanf一样，但我们常用于windows终端的关闭延缓

## 大小端

大端（存储）模式：是指数据的低位字节内容保存在内存的高地址处，而数据的高位字节内容，保存在内存的低地址处。
 小端（存储）模式：是指数据的低位字节内容保存在内存的低地址处，而数据的高位字节内容，保存在内存的高地址处

## 整数的表达：

    %d：正常表达方式

    %#X / %#x / %x: 以16进制形式表达

    %#o / %o：以8进制形式表达

    %ld：表示long

    %lld：表示long long

    %u：表述unsigned int

    %c：表示字符型


## 特殊转义

    \a: 报警

    \f:换页

    \\\：打印\

## 溢出z整数

### 整数溢出

    int（32位）：

    0~7FFFFFFF (从零开始递增)

    80000000~FFFFFFFF(从最小值开始递增，逐渐到达-1)


    当在7FFFFFFF去加1，便会到达最小值80000000（上溢） ;

    挡在80000000减1,便会到达最大值（下溢）

float：

    上溢（数值过大，最终显示inf）

    下溢（精度丢失）


## RPId32

    主要定义在stdtypes.h这个头文件中。主要作用时替换"%d"中的d。

用法如下： printf("word = %" RPId32 "\n", m);
